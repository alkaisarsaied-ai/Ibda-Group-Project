# Ibda'a Group - Professional Company Profile

موقع تعريفي متكامل لشركة "إبداع جروب" للحلول الرقمية، تم بناؤه كمشروع نهائي يجمع بين التصميم العصري والبرمجة التفاعلية.

## 🚀 رابط المعاينة المباشر (Live Demo)
[اضغط هنا لزيارة الموقع](https://charming-bombolone-93f97e.netlify.app/)

## ✨ المميزات الرئيسية (Key Features)
* **Luxury Dark Mode:** نظام تبديل ألوان ذكي يتيح لليوزر الاختيار بين الوضع الفاتح (أبيض/أزرق) والوضع الليلي الفخم (أسود/ذهبي).
* **Fully Responsive:** الموقع متوافق تماماً مع جميع الشاشات (موبايل، تابلت، كمبيوتر).
* **Multi-Page Structure:** يحتوي على 4 صفحات أساسية (Home, About, Services, Contact).
* **Modern UI:** استخدام انتقالات ناعمة (Transitions) وتأثيرات حركية عند التفاعل مع العناصر.

## 🛠️ التقنيات المستخدمة (Built With)
* **HTML5:** لبناء هيكل الصفحات بشكل دلالي (Semantic HTML).
* **Tailwind CSS:** لتنسيق الواجهات بسرعة واحترافية عالية.
* **JavaScript:** لبرمجة منطق تبديل الثيم (Dark/Light Mode) وحفظ اختيار المستخدم.
* **FontAwesome:** للأيقونات الاحترافية.
* **Google Fonts:** استخدمنا خط (Inter) لإعطاء لمسة عصرية للنصوص.
